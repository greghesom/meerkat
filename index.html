Meerkat - Sequence Diagram Visualizer

Description:

Arrow Types

Solid arrow (->>, ->) - Synchronous
Dashed arrow (-->>, -->) - Asynchronous

API Path Annotations

@path(GET /api/users) - Annotate with HTTP method and path
Supported methods: GET, POST, PUT, DELETE, PATCH

Request Type Annotations

@type(JSON) - Specify the protocol/format
Standard types: JSON, SOAP, XML, RFC_SAP, GraphQL, gRPC, BINARY
Custom types are also supported with a default style

Sync/Async Annotations

@sync - Mark as synchronous (solid line)
@async - Mark as asynchronous (dashed line + clock icon)
@timeout(100ms) - Expected response time
@queue(user-events) - Message queue name

Payload Annotations

@request{name: string, email: string} - Request body schema hint
@response{id: uuid, created_at: timestamp} - Response body schema hint
Supports inline schemas and external references (URLs or anchors)

Flow Annotations

%%flow happy_path "Happy Path" #22C55E - Define a flow with optional color
@flow(happy_path) - Assign message to a flow
@flow(happy_path, error_flow) - Shared message (multiple flows)
Use the flow selector above to filter by active flow

Adding New Diagrams

Create a new JSON file in the diagrams/ folder and add it to diagrams/config.json
